getAnimationFrame(callback) → {number}

setTitle(summary, newTitle)

Set the title attribute of an element if it doesn't already have one. This function checks if the specified element already has a 'title' attribute. If it does, it does nothing. If not, it sets the 'title' attribute to the provided newTitle.
Parameters:
Name Type Description
summary Element The element for which to set the 'title' attribute.
